here's a little love website for my cute girlfriend.

it has three sections:

* 💌 letters — Write personal letters with a recipient name, your message, and a signature. All sent letters are saved and can be re-read by clicking on them.
* 📷 photos — Upload photos to a shared memory wall, each with an optional sweet caption.
* 🌸 flowers — Build a virtual bouquet by tapping flowers from the picker, add a note, and send it. All bouquets are saved.

ps. there are falling petals in the background, a confetti burst every time you "send" something.
